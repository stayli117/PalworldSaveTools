Funktionsübersicht

Durchsuchen und verwalten Sie Artikelbestände und Arbeitsfreunde für alle Basen in allen Gilden. Wechseln Sie zwischen zwei Unterregisterkarten: Inventory (Artikel in Containern anzeigen/bearbeiten) und Base Pals (Arbeitspals anzeigen/verwalten, die der ausgewählten Basis zugewiesen sind).

Grundfunktionen (Linksklick und Tasten)

  • Klicken Sie auf die Schaltfläche Gilde, um ein durchsuchbares Popup zu öffnen und wählen Sie eine Gilde aus. Klicken Sie dann auf die Schaltfläche Basis, um eine Basis auszuwählen.
  • Klicken Sie auf einen Container in der Containerliste (linker Bereich), um seine Elemente im Raster anzuzeigen.
  • Klicken Sie auf die Registerkartenschaltflächen Inventar oder Base Pals, um die Ansicht zu wechseln.
  • Verwenden Sie die Schaltfläche Item, um gildenübergreifende Gegenstandsoperationen zu öffnen (Gegenstände in allen Gilden suchen/entfernen).
  • Verwenden Sie die Schaltfläche Struktur zum Löschen einer gildenübergreifenden Struktur.
  • Die Schaltfläche Basen suchen durchsucht alle Basen nach Containern, die einem Artikelfilter entsprechen.
  • Verwenden Sie das Scrollrad im Base Pals-Raster, um durch die Seiten zu navigieren, wenn eine Basis mehr als 30 aktive Pals hat.

Erweiterte Verknüpfungen (Doppelklick-Interaktionen)

  • Doppelklicken Sie auf einen Gegenstand im Inventarraster, um es sofort aus dem Steckplatz zu entfernen (keine Bestätigung).
  • Doppelklicken Sie auf einen leeren Steckplatz, um den Dialog zum Hinzufügen von Elementen zu öffnen für diesen Steckplatz.
  • Doppelklicken Sie auf einen Pal-Slot in Base Pals: Wenn der Slot einen Pal hat, wird er direkt gelöscht (keine Bestätigung); Wenn leer, wird das Dialogfeld „Neuen Pal hinzufügen“ geöffnet.
  • Doppelklicken Sie auf einen Freund in der Suchliste der Gegenstandsdialoge, um Container zu finden, die diesen Gegenstand enthalten.

Versteckte Menüs (Rechtsklick-Kontext)

  • Klicken Sie mit der rechten Maustaste auf einen Container in der Liste für: Element hinzufügen (Debug), Container löschen, Container-Slots ändern, Container löschen.
  • Klicken Sie mit der rechten Maustaste auf einen Artikel im Raster für: Menge bearbeiten, Artikel entfernen, Behälter leeren.
  • Klicken Sie mit der rechten Maustaste auf einen leeren Rasterplatz für: Element hinzufügen, Container löschen.
  • Klicken Sie mit der rechten Maustaste auf einen Kumpel-Slot mit Daten für: Boss/Glück/Erwacht/DNA-Schalter, Sperrstufe (4 Stufen), Max. Alle Statistiken, Alle Bewegungen lernen, Erlernte Fertigkeiten, Bulk-Sync-Kumpel, Löschen.
  • Klicken Sie mit der rechten Maustaste auf einen leeren Pal-Slot für: Neuen Pal hinzufügen.

Power-User-Tipp

Bewegen Sie den Mauszeiger über einen Pal-Slot in Base Pals, um eine Vorschau seiner Statistiken im Infofenster anzuzeigen. Die Paginierung unten zeigt die aktuelle Seite der aktiven Pals – verwenden Sie zum Navigieren die Schaltflächen „Zurück/Weiter“ oder das Scrollrad.
