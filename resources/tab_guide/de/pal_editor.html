Funktionsübersicht

Deep editing interface for any Pal owned by any player. Durchsuchen Sie nach Gruppe (aktive Truppe) und Palbox (Lager) geordnete Freunde mit voller Kontrolle über Statistiken, IVs, aktive Fertigkeiten, passive Eigenschaften, Arbeitseignungen, Rang, Seelen-Upgrades, Erscheinungsflaggen (Boss/Lucky/Erwacht/DNA) und mehr. An info panel on the right shows all editable details.

Grundfunktionen (Linksklick und Tasten)

  • Klicken Sie auf Spieler auswählen..., um einen Spieler aus dem durchsuchbaren Popup auszuwählen (zum Filtern eingeben, zum Auswählen die Eingabetaste eingeben).
  • Klicken Sie auf einen Party- oder Palbox-Slot, um einen Pal zum Bearbeiten auszuwählen – das Infofeld wird mit allen bearbeitbaren Feldern gefüllt.
  • Click toggle buttons in the info panel header: Gender, Boss Alpha, Lucky/Shiny, Awakened, Imported/DNA, Max All Stats.
  • Click the Favorite star to cycle through lock levels (0–3).
  • Click label fields to edit inline via input dialogs: Name (nickname), Level (1–80), Trust Rank (0–10).
  • Click IV labels (HP/Attack/Defense) to set 0–100.
  • Click Soul labels (HP/Attack/Defense/Craft Speed) to set 0–20.
  • Click active skill slots to open the Skill Picker.
  • Click passive slots to open the Passive Picker.
  • Click work suitability values to set their level (0–10).
  • Navigate between Palbox pages using the ← Previous / Next → buttons or mouse wheel on the grid.

Advanced Shortcuts (Double-Click Interactions)

  • Doppelklicken Sie auf einen Party- oder Palbox-Slot, der einen Pal hat, um ihn schnell zu löschen (keine Bestätigung – löst die gleiche „delete_direct“-Aktion wie das Kontextmenü aus).
  • Double-click an empty Party or Palbox slot to open the Add New Pal creation dialog.
  • Double-click a Pal in the Add New Pal list to immediately create it.
  • Double-click the FramelessDialog title bar to toggle maximize.

Versteckte Menüs (Rechtsklick-Kontext)

  • Klicken Sie mit der rechten Maustaste auf einen beliebigen Party- oder Palbox-Slot mit einem Kumpel, um Folgendes zu erhalten: Boss Alpha umschalten, Lucky/Shiny umschalten, Erwacht umschalten, Importiert/DNA umschalten, Stufe sperren (Untermenü: 4 Stufen), Alle Statistiken maximieren, Alle Bewegungen lernen, Dialogfeld „Erlernte Fertigkeiten“, Bulk-Synchronisierungspartner, Löschen.
  • Klicken Sie mit der rechten Maustaste auf einen leeren Steckplatz für: Neuen Freund hinzufügen.
  • Right-click the Pal portrait in the info panel for the same expanded context menu.

Power-User-Tipp

Tastaturkürzel beschleunigen die Bearbeitung: Q = vorherige Palbox-Seite, E = nächste Palbox-Seite, F = Fokus und Scrollen zum Infobereich, C = Fähigkeitsansicht umschalten. Bewegen Sie den Mauszeiger über einen Slot, um eine Vorschau seines Pal im Infofeld anzuzeigen, ohne ihn auszuwählen. Verwenden Sie Bulk Sync (aus dem Kontextmenü), um Statistiken/Fähigkeiten/Passive von einem Pal auf mehrere andere gleichzeitig zu kopieren.
