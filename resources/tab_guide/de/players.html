Funktionsübersicht

Durchsuchbare Tabelle aller Spieler im Spielstand. In den Spalten werden Spielername, Zuletzt gesehen, Level, Anzahl der Freunde, UID, Gildenname, Gilden-ID und Gildenlevel angezeigt. Verwenden Sie die Massenaktionsschaltflächen unten für die Verwaltung von Gegenständen, Freunden und Technologien für mehrere Spieler.

Grundfunktionen (Linksklick und Tasten)

  • Geben Sie die Suchleiste ein, um Spieler in Echtzeit zu filtern – jede Spaltenübereinstimmung wird angezeigt.
  • Klicken Sie auf eine Zeile, um sie auszuwählen.
  • Klicken Sie auf Bulk Item Management, um das Dialogfeld „Player Item Action“ (Elemente für ausgewählte Spieler hinzufügen/entfernen) zu öffnen.
  • Klicken Sie auf Bulk Pal Management für Pal-Operationen.
  • Klicken Sie auf Bulk Technology Management für freigeschaltete Technologien.

Erweiterte Verknüpfungen (Doppelklick-Interaktionen)

  • Ein Doppelklick auf eine Spielerzeile gibt das Signal item_double_clicked aus.
  • Allerdings ist im aktuellen Code kein Standardhandler verbunden – Doppelklick hat derzeit keine sichtbare Auswirkung.
  • (Für zukünftige Verwendung vorgesehen.)

Versteckte Menüs (Rechtsklick-Kontext)

  • Klicken Sie mit der rechten Maustaste auf eine Spielerzeile für: Ausschluss hinzufügen/entfernen, Spieler löschen, Spieler umbenennen, Anzeigekäfig entsperren, Zeitstempel zurücksetzen, Container-IDs aktualisieren, Alle Technologien freischalten, Tech Points bearbeiten, Spielerstatistiken bearbeiten, Spielerstufe festlegen.
  • Enthält außerdem Aktionen auf Gildenebene für die Gilde des Spielers: Gildenanführer ernennen, Gilde löschen, Gilde umbenennen, Gesamte Laborforschung freischalten, Gildenstufe festlegen, Basis in Gilde importieren.

Power-User-Tipp

Sortieren Sie eine beliebige Spalte, indem Sie auf deren Überschrift klicken. Als Schlüssel für Ausschluss- und Löschvorgänge wird die UID-Spalte (Spaltenindex 4) verwendet. Verwenden Sie Menü → Funktionen → Inaktive Spieler löschen, nachdem Sie die zuletzt gesehenen Daten überprüft haben, um verlassene Spieler in großen Mengen zu bereinigen.
