Docs & Wiki Reference

The Docs tab provides an in-app reference guide to Palworld game data. Browse categories to inspect Pals, Items, Buildings, Skills, Elements, Work Suitability, and Technologies without leaving the tool.

Wiki Categories
  • Pals — Complete pal database with stats, element types, work suitability, partner skills, and per-pal skill learnsets (level-up & egg moves).
  • Items — All items with category, rarity, price, weight, stack size, and descriptions.
  • Buildings — Structure data including stats, rank, materials, and base requirements.
  • Active Skills — Skill list with element, power, cooldown, hit power rate, max hits, and hit interval values extracted from game blueprints.
  • Passive Skills — Passive skill database with rank, description, and effect values.
  • Technologies — Technology tree with cost, level cap, unlockable items and buildings.
  • Elements — Element reference with color codes and associated pals.
  • Work Suitability — Work type reference and which pals perform each type.
Navigation
  • Select a category from the left sidebar to browse its entries.
  • Click any entry in the list to view its details in the right panel.
  • Use the search bar to filter entries by name or internal code name.

Note

All game data is sourced from the latest game exports. Data is loaded on-demand per category for fast startup. Skill hit data (WazaPowerRate, MaxHitNum, HitInterval) is extracted directly from game blueprint files.
