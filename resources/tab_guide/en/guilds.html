Functional Overview

Two-panel view: top panel lists all guilds with columns for Guild Name, Guild ID, and Guild Level. Bottom panel shows the members of the selected guild with columns for Member Name, Last Seen, Level, Pals, and UID.

Basic Operations (Left-Click & Buttons)

  • Type in the search bar to filter guilds by any column.
  • Click a guild row to select it — the bottom panel automatically populates with that guild's member players.
  • Click a member row to view their details.

Advanced Shortcuts (Double-Click Interactions)

  • Double-clicking a guild or member row emits the item_double_clicked signal.
  • No default handler is connected — double-click currently has no visible effect on either panel.

Hidden Menus (Right-Click Context)

  • Right-click a guild row for: Add/Remove Exclusion, Delete Guild, Rename Guild, Set Guild Level, Unlock All Lab Research, Export Bases for Guild, Import Bases (Multi-File), Move Selected Player to Selected Guild.
  • Right-click a member row for: Make Guild Leader, Unlock All Lab Research for Guild, Add/Remove Player Exclusion, Delete Player, Rename Player, Reset Timestamp, Set Player Level.

Power-User Tip

Use Menu → Functions → Delete Empty Guilds to remove guilds with zero members in one click. The guild ID (column index 1) is the key reference used by exclusion lists and deletion operations.
