Welcome to Palworld Save Tools V2

This is Palworld Save Tools V2 — a streamlined edition focused on the core features you need most. Everything is designed to be fast, intuitive, and easy to navigate.

Browse the sections below or use the Table of Contents above to jump straight to any guide.

Quick Start
  • Click Menu → Load Save or drag-and-drop a .sav file onto the window to get started.
  • Use the tabs (Map, Tools, Players, Guilds, Bases, etc.) to explore and edit your save data.
  • Right-click almost anything for contextual actions — deletions, edits, exports, and more.
  • Save your changes regularly via Menu → Save Changes. Backups are created automatically.
What's Available
  • Map Viewer — Interactive world map with base and player markers, zone drawing, and calibration.
  • Tools — Save conversion, character transfer, host fix, slot injector, and cleanup functions.
  • Base Inventory — Browse and manage items and working Pals across all bases.
  • Player Inventory — Full inventory, key items, stats, and equipment management.
  • Pal Editor — Deep pal editing: stats, IVs, skills, passives, work suitabilities, and more.
  • Players — Player table with bulk item, pal, and technology management.
  • Guilds — Two-panel guild and member management with full context actions.
  • Bases — Base camp table with export, import, clone, and radius adjustment.
  • Exclusions — Protection lists to safeguard players, guilds, and bases from cleanup.
  • Docs — In-app game data reference: Pals, Items, Skills, Buildings, Elements, and more.
