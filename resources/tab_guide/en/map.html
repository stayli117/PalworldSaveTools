Functional Overview

Interactive world map showing base markers (house icon), player markers (person icon), and exclusion zones. Click markers to inspect bases/players, right-click for management actions, and use the sidebar tree to navigate by guild.

Basic Operations (Left-Click & Buttons)

  • Click a marker to select it (highlights with glow effect).
  • Scroll mouse wheel to zoom (1.15x per step, range 1.0x–30.0x).
  • Click-and-drag empty space to pan.
  • Toggle map overlays via toolbar: Bases, Players, Radius Rings, and Exclusion Zones buttons.
  • Switch between World Map and Tree Map views.
  • The search bar filters markers by guild or player name.
  • The Bases/Players tab buttons on the left switch between base tree and player tree panels.
  • Click Calibrate to enter calibration mode for precise coordinate alignment.

Advanced Shortcuts (Double-Click Interactions)

  • Double-click any base or player marker to zoom in and animate the map view to that marker at 26.0x zoom while updating the info panel with its details.
  • Double-click a zone item (exclusion zone drawn on the map) to open a rename dialog.
  • In zone-drawing mode with Rect selected: first double-click sets corner A, second double-click sets corner B and creates the zone.
  • In Polygon mode with at least 3 points: double-click closes and creates the polygon zone.

Hidden Menus (Right-Click Context)

  • Right-click a base marker for: Delete Base, Export Base, Adjust Radius.
  • Right-click a player marker for: Delete Player, Rename Player, Unlock Viewing Cage, Unlock Technologies.
  • Right-click empty map space for: Import Base, Draw Zones, Clear All Zones, Export Zones, Import Zones.
  • Right-click items in the base/player tree for additional actions including guild-level operations (Rename Guild, Delete Guild, Export/Import Bases for Guild, Update Container IDs).
  • Right-click a zone on the map for: Delete Zone (with confirmation), Rename Zone.

Power-User Tip

Press Escape while in zone-drawing mode to cancel and exit immediately. Use calibration mode (Calibrate button) to precisely align the map with game coordinates by left-clicking known locations and right-clicking to undo.
