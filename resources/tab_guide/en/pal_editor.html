Functional Overview

Deep editing interface for any Pal owned by any player. Browse Pals organized by Party (active squad) and Palbox (storage), with full control over stats, IVs, active skills, passive traits, work suitabilities, rank, soul upgrades, appearance flags (Boss/Lucky/Awakened/DNA), and more. An info panel on the right shows all editable details.

Basic Operations (Left-Click & Buttons)

  • Click Select Player... to choose a player from the searchable popup (type to filter, Enter to select).
  • Click a Party or Palbox slot to select a Pal for editing — the info panel populates with all editable fields.
  • Click toggle buttons in the info panel header: Gender, Boss Alpha, Lucky/Shiny, Awakened, Imported/DNA, Max All Stats.
  • Click the Favorite star to cycle through lock levels (0–3).
  • Click label fields to edit inline via input dialogs: Name (nickname), Level (1–80), Trust Rank (0–10).
  • Click IV labels (HP/Attack/Defense) to set 0–100.
  • Click Soul labels (HP/Attack/Defense/Craft Speed) to set 0–20.
  • Click active skill slots to open the Skill Picker.
  • Click passive slots to open the Passive Picker.
  • Click work suitability values to set their level (0–10).
  • Navigate between Palbox pages using the ← Previous / Next → buttons or mouse wheel on the grid.

Advanced Shortcuts (Double-Click Interactions)

  • Double-click a Party or Palbox slot that has a Pal to quick-delete it (no confirmation — fires the same 'delete_direct' action as the context menu).
  • Double-click an empty Party or Palbox slot to open the Add New Pal creation dialog.
  • Double-click a Pal in the Add New Pal list to immediately create it.
  • Double-click the FramelessDialog title bar to toggle maximize.

Hidden Menus (Right-Click Context)

  • Right-click any Party or Palbox slot with a Pal for: Boss Alpha toggle, Lucky/Shiny toggle, Awakened toggle, Imported/DNA toggle, Lock Level (submenu: 4 levels), Max All Stats, Learn All Moves, Learnt Skills dialog, Bulk Sync Pal, Delete.
  • Right-click an empty slot for: Add New Pal.
  • Right-click the Pal portrait in the info panel for the same expanded context menu.

Power-User Tip

Keyboard shortcuts speed up editing: Q = previous Palbox page, E = next Palbox page, F = focus and scroll to the info panel, C = toggle skills view. Hover over a slot to preview its Pal in the info panel without selecting it. Use Bulk Sync (from context menu) to copy stats/skills/passives from one Pal to multiple others at once.
