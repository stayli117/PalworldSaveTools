Documentación y Referencia Wiki

La pestaña Documentación proporciona una guía de referencia integrada sobre los datos del juego Palworld. Explore categorías para inspeccionar Pals, Objetos, Edificios, Habilidades, Elementos, Idoneidad Laboral y Tecnologías sin salir de la herramienta.

Categorías Wiki
  • Pals — Base de datos completa de Pals con estadísticas, tipos de elemento, idoneidad laboral, habilidades de compañero y conjuntos de habilidades aprendibles (subida de nivel y huevo).
  • Objetos — Todos los objetos con categoría, rareza, precio, peso, tamaño de pila y descripciones.
  • Edificios — Datos de estructuras incluyendo estadísticas, rango, materiales y requisitos de base.
  • Habilidades Activas — Lista de habilidades con elemento, poder, enfriamiento, tasa de poder de impacto, impactos máximos y valores de intervalo extraídos de los planos del juego.
  • Habilidades Pasivas — Base de datos de habilidades pasivas con rango, descripción y valores de efecto.
  • Tecnologías — Árbol tecnológico con costo, nivel requerido y objetos/edificios desbloqueables.
  • Elementos — Referencia de elementos con códigos de color y Pals asociados.
  • Idoneidad Laboral — Referencia de tipos de trabajo y qué Pals realizan cada tipo.
Navegación
  • Seleccione una categoría en la barra lateral izquierda para explorar sus entradas.
  • Haga clic en cualquier entrada de la lista para ver sus detalles en el panel derecho.
  • Use la barra de búsqueda para filtrar entradas por nombre o código interno.

Nota

Todos los datos del juego provienen de las últimas exportaciones del juego. Los datos se cargan bajo demanda por categoría para un inicio rápido. Los datos de impacto (WazaPowerRate, MaxHitNum, HitInterval) se extraen directamente de los archivos de plano del juego.
