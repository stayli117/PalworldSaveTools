Descripción funcional

Tabla de búsqueda de todos los jugadores guardados. Las columnas muestran el nombre del jugador, el último visto, el nivel, el número de amigos, el UID, el nombre del gremio, el ID del gremio y el nivel del gremio. Utilice los botones de acción masiva en la parte inferior para administrar elementos, amigos y tecnología en varios jugadores.

Operaciones básicas (clic izquierdo y botones)

  • Escriba en la barra de búsqueda para filtrar jugadores en tiempo real; se mostrará cualquier coincidencia de columna.
  • Haga clic en una fila para seleccionarla.
  • Haga clic en Gestión masiva de elementos para abrir el cuadro de diálogo Acción de elementos del jugador (agregar/eliminar elementos entre los jugadores seleccionados).
  • Haga clic en Gestión masiva de Pal para operaciones Pal.
  • Haga clic en Gestión de tecnología masiva para desbloquear tecnología.

Atajos avanzados (interacciones de doble clic)

  • Al hacer doble clic en una fila de jugador se emite la señal item_double_clicked.
  • Sin embargo, no hay ningún controlador predeterminado conectado en el código actual; actualmente, hacer doble clic no tiene ningún efecto visible.
  • (Diseñado para uso futuro.)

Menús ocultos (contexto de clic derecho)

  • Haga clic derecho en una fila de jugador para: Agregar/eliminar exclusión, eliminar jugador, cambiar nombre de jugador, desbloquear jaula de visualización, restablecer marca de tiempo, actualizar ID de contenedores, desbloquear todas las tecnologías, editar puntos tecnológicos, editar estadísticas de jugador, establecer nivel de jugador.
  • También incluye acciones a nivel de gremio para el gremio del jugador: Convertir en líder de gremio, eliminar gremio, cambiar nombre de gremio, desbloquear todas las investigaciones de laboratorio, establecer nivel de gremio, importar base a gremio.

Consejo para usuarios avanzados

Ordene cualquier columna haciendo clic en su encabezado. La columna UID (índice de columna 4) se utiliza como clave para las operaciones de exclusión y eliminación. Utilice Menú → Funciones → Eliminar jugadores inactivos después de verificar las fechas de última vista para limpiar los jugadores abandonados en masa.
