Aperçu fonctionnel

Parcourez et gérez les inventaires d'objets et les copains de travail pour toutes les bases de toutes les guildes. Basculez entre deux sous-onglets : Inventaire (afficher/modifier les articles dans les conteneurs) et Base Pals (afficher/gérer les amis de travail affectés à la base sélectionnée).

Opérations de base (clic gauche et boutons)

  • Cliquez sur le bouton Guild pour ouvrir une fenêtre contextuelle de recherche et sélectionnez une guilde, puis cliquez sur le bouton Base pour sélectionner une base.
  • Cliquez sur un conteneur dans la liste des conteneurs (panneau de gauche) pour afficher ses éléments dans la grille.
  • Cliquez sur les boutons de l'onglet Inventory ou Base Pals pour changer de vue.
  • Utilisez le bouton Item pour ouvrir les opérations sur les objets entre les guildes (trouver/supprimer des objets dans toutes les guildes).
  • Utilisez le bouton Structure pour supprimer la structure inter-guilde.
  • Le bouton Rechercher des bases recherche dans toutes les bases les conteneurs correspondant à un filtre d'articles.
  • Utilisez la molette de défilement de la grille Base Pals pour parcourir les pages lorsqu'une base compte plus de 30 Pals actifs.

Raccourcis avancés (interactions par double-clic)

  • Double-cliquez sur un objet dans la grille d'inventaire pour le retirer immédiatement de l'emplacement (pas de confirmation).
  • Double-cliquez sur un emplacement vide pour ouvrir la boîte de dialogue d'ajout d'élément pour cet emplacement.
  • Double-cliquez sur un slot Pal dans Base Pals : si le slot possède un Pal, il est supprimé directement (pas de confirmation) ; s'il est vide, il ouvre la boîte de dialogue Ajouter un nouveau Pal.
  • Double-cliquez sur un ami dans la liste de recherche des boîtes de dialogue d'objets pour trouver les conteneurs contenant cet objet.

Menus cachés (contexte du clic droit)

  • Cliquez avec le bouton droit sur un conteneur dans la liste pour : Ajouter un élément (débogage), Effacer le conteneur, Modifier les emplacements du conteneur, Supprimer le conteneur.
  • Cliquez avec le bouton droit sur un élément dans la grille pour : Modifier la quantité, Supprimer l'article, Effacer le conteneur.
  • Cliquez avec le bouton droit sur un emplacement de grille vide pour : Ajouter un élément, Effacer le conteneur.
  • Cliquez avec le bouton droit sur un emplacement Pal avec les données pour : bascules Boss/Lucky/Éveillé/ADN, niveau de verrouillage (4 niveaux), toutes les statistiques maximales, apprendre tous les mouvements, compétences acquises, synchronisation groupée avec Pal, supprimer.
  • Cliquez avec le bouton droit sur un emplacement Pal vide pour : Ajouter un nouveau Pal.

Conseil d'utilisateur expérimenté

Survolez un emplacement Pal dans Base Pals pour prévisualiser ses statistiques dans le panneau d'informations. La pagination en bas montre la page actuelle des copains qui travaillent — utilisez les boutons précédent/suivant ou la molette de défilement pour naviguer.
