Aperçu fonctionnel

Tableau consultable de tous les camps de base dans la sauvegarde. Les colonnes indiquent l'ID de base, l'ID de guilde, le nom de guilde et le niveau de guilde. Le menu contextuel du clic droit permet une gestion complète de la base : suppression, ajustement du rayon, importation/exportation, clonage et opérations de guilde.

Opérations de base (clic gauche et boutons)

  • Tapez dans la barre de recherche pour filtrer les bases par n'importe quelle colonne.
  • Cliquez sur une ligne de base pour la sélectionner.

Raccourcis avancés (interactions par double-clic)

  • Un double-clic sur une ligne de base émet le signal item_double_clicked.
  • Aucun gestionnaire par défaut n'est connecté — le double-clic n'a actuellement aucun effet visible.

Menus cachés (contexte du clic droit)

  • Cliquez avec le bouton droit sur une ligne de base pour : Ajouter/Supprimer une exclusion, Supprimer la base (nécessite l'ID de base et l'ID de guilde), Renommer la guilde, Définir le niveau de guilde, Exporter la base, Ajuster le rayon, Importer la base, Cloner la base.
  • Notez que certaines actions opèrent sur la guilde associée à la base plutôt que sur la base elle-même.

Conseil d'utilisateur expérimenté

L'ID de base (index de colonne 0) et l'ID de guilde (index de colonne 1) sont tous deux nécessaires pour la suppression — le menu contextuel transmet les deux. Utilisez le chemin Menu → Fonctions pour le nettoyage de base en masse, comme Supprimer les bases inactives et Supprimer les objets cartographiques non basés sur la base.
