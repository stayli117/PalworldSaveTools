Documentation & Référence Wiki

L'onglet Documentation fournit un guide de référence intégré aux données du jeu Palworld. Parcourez les catégories pour consulter les Pals, Objets, Bâtiments, Compétences, Éléments, Aptitudes de Travail et Technologies sans quitter l'outil.

Catégories Wiki
  • Pals — Base de données complète des Pals avec statistiques, types d'élément, aptitudes de travail, compétences de partenaire et ensembles de compétences apprises (niveau et œuf).
  • Objets — Tous les objets avec catégorie, rareté, prix, poids, taille de pile et descriptions.
  • Bâtiments — Données des structures incluant statistiques, rang, matériaux et exigences de base.
  • Compétences Actives — Liste des compétences avec élément, puissance, temps de recharge, taux de puissance de coup, coups maximum et valeurs d'intervalle extraites des plans du jeu.
  • Compétences Passives — Base de données des compétences passives avec rang, description et valeurs d'effet.
  • Technologies — Arbre technologique avec coût, niveau requis et objets/bâtiments débloquables.
  • Éléments — Référence des éléments avec codes couleur et Pals associés.
  • Aptitudes de Travail — Référence des types de travail et quels Pals effectuent chaque type.
Navigation
  • Sélectionnez une catégorie dans la barre latérale gauche pour parcourir ses entrées.
  • Cliquez sur une entrée dans la liste pour voir ses détails dans le panneau de droite.
  • Utilisez la barre de recherche pour filtrer les entrées par nom ou code interne.

Note

Toutes les données du jeu proviennent des dernières exportations du jeu. Les données sont chargées à la demande par catégorie pour un démarrage rapide. Les données de coup (WazaPowerRate, MaxHitNum, HitInterval) sont extraites directement des fichiers blueprint du jeu.
