Aperçu fonctionnel

Vue à deux panneaux : le panneau supérieur répertorie toutes les guildes avec des colonnes pour le nom de guilde, l'identifiant de guilde et le niveau de guilde. Le panneau inférieur affiche les membres de la guilde sélectionnée avec des colonnes pour le nom du membre, la dernière visite, le niveau, les amis et l'UID.

Opérations de base (clic gauche et boutons)

  • Tapez dans la barre de recherche pour filtrer les guildes par n'importe quelle colonne.
  • Cliquez sur une ligne de guilde pour la sélectionner : le panneau inférieur se remplit automatiquement avec les joueurs membres de cette guilde.
  • Cliquez sur une ligne de membre pour afficher ses détails.

Raccourcis avancés (interactions par double-clic)

  • Double-cliquer sur une ligne de guilde ou de membre émet le signal item_double_clicked.
  • Aucun gestionnaire par défaut n'est connecté — le double-clic n'a actuellement aucun effet visible sur aucun des deux panneaux.

Menus cachés (clic droit contextuel)

  • Cliquez avec le bouton droit sur une ligne de guilde pour : ajouter/supprimer une exclusion, supprimer une guilde, renommer une guilde, définir le niveau de guilde, débloquer toutes les recherches en laboratoire, exporter des bases pour une guilde, importer des bases (multi-fichiers), déplacer le joueur sélectionné vers la guilde sélectionnée.
  • Cliquez avec le bouton droit sur une ligne de membre pour : devenir chef de guilde, débloquer toutes les recherches de laboratoire pour la guilde, ajouter/supprimer une exclusion de joueur, supprimer un joueur, renommer un joueur, réinitialiser l'horodatage, définir le niveau du joueur.

Conseil d'utilisateur expérimenté

Utilisez Menu → Fonctions → Supprimer les guildes vides pour supprimer les guildes sans membre en un seul clic. L'ID de guilde (index de colonne 1) est la référence clé utilisée par les listes d'exclusion et les opérations de suppression.
