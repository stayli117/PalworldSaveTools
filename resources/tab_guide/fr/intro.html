Bienvenue dans Palworld Save Tools V2

Il s'agit de Palworld Save Tools V2 — une édition simplifiée axée sur les fonctionnalités de base dont vous avez le plus besoin. Tout est conçu pour être rapide, intuitif et facile à naviguer.

Parcourez les sections ci-dessous ou utilisez la Table des matières ci-dessus pour accéder directement à n'importe quel guide.

Démarrage rapide
  • Cliquez sur Menu → Charger Save ou faites glisser et déposez un fichier .sav sur la fenêtre pour commencer.
  • Utilisez les onglets (Carte, Outils, Joueurs, Guildes, Bases, etc.) pour explorer et modifier vos données de sauvegarde.
  • Cliquez avec le bouton droit sur presque tout pour effectuer des actions contextuelles : suppressions, modifications, exportations, etc.
  • Enregistrez régulièrement vos modifications via Menu → Enregistrer les modifications. Les sauvegardes sont créées automatiquement.
Ce qui est disponible
  • Map Viewer — Carte du monde interactive avec marqueurs de base et de joueur, dessin de zone et étalonnage.
  • Tools — Fonctions de conversion d'enregistrement, de transfert de caractères, de correctif d'hôte, d'injecteur d'emplacement et de nettoyage.
  • Inventaire de base — Parcourez et gérez les objets et les amis de travail dans toutes les bases.
  • Inventaire du joueur — Inventaire complet, éléments clés, statistiques et gestion de l'équipement.
  • Pal Editor — Édition approfondie par des amis : statistiques, IV, compétences, passifs, aptitudes au travail, et plus encore.
  • Players — Table des joueurs avec gestion des objets en vrac, des amis et de la technologie.
  • Guildes — Gestion des guildes et des membres à deux panneaux avec actions contextuelles complètes.
  • Bases — Table de camp de base avec réglage de l'exportation, de l'importation, du clonage et du rayon.
  • Exclusions — Listes de protection pour protéger les joueurs, les guildes et les bases du nettoyage.
  • Documentation — Référence des données du jeu intégrée : Pals, Objets, Compétences, Bâtiments, Éléments et plus.
