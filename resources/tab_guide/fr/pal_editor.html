Aperçu fonctionnel

Interface d'édition approfondie pour n'importe quel copain appartenant à n'importe quel joueur. Parcourez les amis organisés par groupe (équipe active) et Palbox (stockage), avec un contrôle total sur les statistiques, les IV, les compétences actives, les traits passifs, les aptitudes au travail, le rang, les améliorations d'âme, les drapeaux d'apparence (Boss/Lucky/Awakened/DNA), et plus encore. Un panneau d'informations sur la droite affiche tous les détails modifiables.

Opérations de base (clic gauche et boutons)

  • Cliquez sur Sélectionner un joueur... pour choisir un joueur dans la fenêtre contextuelle de recherche (tapez pour filtrer, Entrée pour sélectionner).
  • Cliquez sur un emplacement Party ou Palbox pour sélectionner un Pal à modifier — le panneau d'informations se remplit avec tous les champs modifiables.
  • Cliquez sur les boutons bascule dans l'en-tête du panneau d'informations : Sexe, Boss Alpha, Chanceux/Brillant, Éveillé, Importé/ADN, Max Toutes les statistiques.
  • Cliquez sur l'étoile Favoris pour parcourir les niveaux de verrouillage (0 à 3).
  • Cliquez sur les champs d'étiquette pour les modifier en ligne via les boîtes de dialogue de saisie : Nom (surnom), Niveau (1 à 80), Rang de confiance (0 à 10).
  • Cliquez sur les étiquettes IV (HP/Attaque/Défense) pour définir 0 à 100.
  • Cliquez sur les étiquettes Soul (HP/Attaque/Défense/Craft Speed) pour définir 0 à 20.
  • Cliquez sur les emplacements de compétences actifs pour ouvrir le sélecteur de compétences.
  • Cliquez sur les emplacements passifs pour ouvrir le sélecteur de passif.
  • Cliquez sur les valeurs d'aptitude au travail pour définir leur niveau (0 à 10).
  • Naviguez entre les pages Palbox à l'aide des boutons ← Précédent / Suivant → ou de la molette de la souris sur la grille.

Raccourcis avancés (interactions par double-clic)

  • Double-cliquez sur un emplacement Party ou Palbox qui a un Pal pour le supprimer rapidement (pas de confirmation - déclenche la même action 'delete_direct' que le menu contextuel).
  • Double-cliquez sur un emplacement Party ou Palbox vide pour ouvrir la boîte de dialogue de création Ajouter un nouveau copain.
  • Double-cliquez sur un ami dans la liste Ajouter un nouvel ami pour le créer immédiatement.
  • Double-cliquez sur la barre de titre FramelessDialog pour activer l'agrandissement.

Menus cachés (contexte du clic droit)

  • Cliquez avec le bouton droit sur n'importe quel emplacement Party ou Palbox avec un Pal pour : bascule Boss Alpha, bascule Chance/Brillant, bascule Éveillé, bascule Importé/ADN, Niveau de verrouillage (sous-menu : 4 niveaux), Toutes les statistiques maximales, Apprendre tous les mouvements, Boîte de dialogue Compétences acquises, Synchronisation groupée Pal, Supprimer.
  • Cliquez avec le bouton droit sur un emplacement vide pour : Ajouter un nouveau copain.
  • Cliquez avec le bouton droit sur le portrait de Pal dans le panneau d'informations pour afficher le même menu contextuel développé.

Conseil d'utilisateur expérimenté

Les raccourcis clavier accélèrent l'édition : Q = page Palbox précédente, E = page Palbox suivante, F = focus et défilement jusqu'au panneau d'informations, C = basculer l'affichage des compétences. Passez la souris sur un emplacement pour prévisualiser son Pal dans le panneau d'informations sans le sélectionner. Utilisez Bulk Sync (à partir du menu contextuel) pour copier les statistiques/compétences/passifs d'un ami vers plusieurs autres à la fois.
