機能概要

すべてのギルドのすべての拠点のアイテム在庫と作業仲間を閲覧および管理します。 2 つのサブタブを切り替えます: Inventory (コンテナ内のアイテムの表示/編集) と Base Pals (選択したベースに割り当てられた作業 Pals の表示/管理)。

基本操作 (左クリックとボタン)

  • ギルド ボタンをクリックして検索可能なポップアップを開き、ギルドを選択してから、ベース ボタンをクリックして拠点を選択します。
  • コンテナ リスト (左側のパネル) でコンテナをクリックすると、そのアイテムがグリッドに表示されます。
  • 在庫 または Base Pals タブ ボタンをクリックしてビューを切り替えます。
  • アイテム ボタンを使用して、ギルド間のアイテム操作 (すべてのギルドにわたるアイテムの検索/削除) を開きます。
  • ギルド間の構造を削除するには、構造ボタンを使用します。
  • ベースの検索 ボタンは、項目フィルターに一致するコンテナーのすべてのベースをスキャンします。
  • 拠点に 30 人を超える作業仲間がいる場合は、拠点仲間グリッドのスクロール ホイールを使用してページを移動します。

高度なショートカット (ダブルクリック操作)

  • インベントリ グリッド内のアイテムをダブルクリックすると、すぐにスロットから削除されます (確認なし)。
  • 空のスロットをダブルクリックすると、そのスロットの項目追加ダイアログが開きます。
  • Base Pals の Pal スロットをダブルクリックします。スロットに Pal がある場合、直接削除されます (確認なし)。空の場合、Add New Pal ダイアログが開きます。
  • アイテム ダイアログの検索リストで仲間をダブルクリックすると、そのアイテムが入っているコンテナが見つかります。

隠しメニュー (コンテキストを右クリック)

  • リスト内のコンテナを右クリックします: 項目の追加 (デバッグ)、コンテナのクリア、コンテナ スロットの変更、コンテナの削除。
  • グリッド内のアイテムを右クリック: 数量の編集、アイテムの削除、コンテナのクリア。
  • 空のグリッド スロットを右クリックして、項目の追加、コンテナのクリアを行います。
  • 次のデータを含むパル スロットを右クリックします: ボス/ラッキー/覚醒/DNA切り替え、ロック レベル (4 レベル)、すべてのステータスの最大化、すべての動きの学習、習得したスキル、一括同期パル、削除。
  • 空の Pal スロットを右クリックして、新しい Pal を追加します。

パワーユーザー向けのヒント

Base Pals の Pal スロットにカーソルを合わせると、情報パネルで統計をプレビューできます。下部のページネーションには、作業中の Pal の現在のページが表示されます。移動するには、前/次ボタンまたはスクロール ホイールを使用します。
