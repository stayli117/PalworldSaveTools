機能概要

保存されているすべてのベースキャンプの検索可能なテーブル。列には、Base ID、Guild ID、Guild Name、および Guild Level が表示されます。右クリックのコンテキスト メニューでは、完全な基本管理 (削除、半径調整、インポート/エクスポート、クローン作成、ギルド操作など) を実行できます。

基本操作 (左クリックとボタン)

  • 検索バーに入力して、任意の列で塩基をフィルタリングします。
  • ベース行をクリックして選択します。

高度なショートカット (ダブルクリック操作)

  • ベース行をダブルクリックすると、item_double_clicked シグナルが生成されます。
  • デフォルトのハンドラーが接続されていません。現在、ダブルクリックしても目に見える効果はありません。

隠しメニュー (コンテキストを右クリック)

  • ベース行を右クリックして、除外の追加/削除、ベースの削除 (ベース ID とギルド ID が必要)、ギルド名の変更、ギルド レベルの設定、ベースのエクスポート、半径の調整、ベースのインポート、ベースのクローン作成を行います。
  • 一部のアクションは、ベース自体ではなく、ベースに関連付けられた ギルド で動作することに注意してください。

パワー ユーザー向けのヒント

削除にはベース ID (列インデックス 0) とギルド ID (列インデックス 1) の両方が必要です。コンテキスト メニューは両方を渡します。非アクティブなベースの削除や非ベース マップ オブジェクトの削除などの一括ベース クリーンアップには、Menu → Functions パスを使用します。
