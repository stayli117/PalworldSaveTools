機能概要

セーブ ファイル管理の中心ハブ — JSON と SAV、GamePass と Steam 形式間の変換、セーブ間での文字の転送、破損したホスト セーブの修正、プレーヤー スロットの挿入を行います。各ツールは、アイコンと説明を含むクリック可能なカードとして表示されます。

基本操作 (左クリックとボタン)

  • いずれかのツール カードをクリックして操作を開始します (セーブの変換、GamePass→Steam の変換、SteamID の変換、マップの復元、スロット インジェクター、セーブの変更、キャラクター転送、ホスト セーブの修正)。
  • ヘッダーの Load Save ボタン、または Menu → File → Load Save パスを使用します。
  • タイトルの下にある保存パス ラベルにより、ファイル エクスプローラーが開き、現在の保存場所が表示されます。
  • 統計バーのクイック ナビゲーション アイコン (プレイヤー、ギルド、拠点、仲間) は、それらのタブに直接ジャンプします。

高度なショートカット (ダブルクリック インタラクション)

  • ツール カードにはダブルクリックの処理はありません。シングルクリックが唯一のトリガーです。
  • [変換オプション]ダイアログ (SAV↔JSON) では、リスト項目のダブルクリックはサポートされていません。代わりにオプション ボタンを使用してください。

隠しメニュー (コンテキストを右クリック)

  • ツール カードには右クリック アクションがありません。すべての機能は左クリックのみです。
  • ただし、Menu ボタン (左上) には、カードとしては利用できない多くのクリーンアップ機能が含まれています: 空のギルドの削除、非アクティブな拠点/プレイヤーの削除、重複プレイヤーの削除、違法な仲間の修正、無効なアイテム/建造物/仲間の削除、ダンジョン/石油採掘装置/侵略者/供給のリセット、すべての建造物の修復、ロック解除プライベートチェストなど。
