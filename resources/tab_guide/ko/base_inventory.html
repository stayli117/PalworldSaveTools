기능 개요

모든 길드의 모든 기지에 대한 아이템 인벤토리와 작업 친구를 탐색하고 관리합니다. 두 개의 하위 탭 사이를 전환합니다: Inventory(컨테이너의 항목 보기/편집) 및 Base Pals(선택한 베이스에 할당된 작업 Pal 보기/관리).

기본 조작(왼쪽 클릭 및 버튼)

  • 길드 버튼을 클릭해 검색 가능한 팝업을 열고 길드를 선택한 후, 거점 버튼을 눌러 거점을 선택하세요.
  • 컨테이너 목록(왼쪽 패널)에서 컨테이너를 클릭하면 그리드에서 해당 항목을 볼 수 있습니다.
  • 보기를 전환하려면 Inventory 또는 Base Pals 탭 버튼을 클릭하세요.
  • 길드 간 아이템 작업을 열려면 아이템 버튼을 사용하세요(모든 길드에서 아이템 찾기/제거).
  • 길드 간 구조를 삭제하려면 구조 버튼을 사용하세요.
  • 기지 찾기 버튼은 항목 필터와 일치하는 컨테이너의 모든 기지를 검색합니다.
  • 베이스에 작동 중인 Pal이 30개 이상 있을 때 Base Pals 그리드의 스크롤 휠을 사용하여 페이지를 탐색하세요.

고급 단축키(더블 클릭 상호 작용)

  • 인벤토리 그리드의 항목을 두 번 클릭하여 슬롯에서 즉시 제거(확인 없음).
  • 빈 슬롯을 두 번 클릭하여 해당 슬롯에 대한 항목 추가 대화상자를 엽니다.
  • Base Pals에서 Pal 슬롯을 두 번 클릭합니다. 슬롯에 Pal이 있는 경우 직접 삭제(확인 없음); 비어 있으면 새 Pal 추가 대화 상자가 열립니다.
  • 항목 대화 상자의 검색 목록에서 친구를 두 번 클릭하면 해당 항목이 들어 있는 컨테이너를 찾을 수 있습니다.

숨겨진 메뉴(마우스 오른쪽 버튼 클릭 컨텍스트)

  • 목록에서 컨테이너를 마우스 오른쪽 버튼으로 클릭합니다: 항목 추가(디버그), 컨테이너 지우기, 컨테이너 슬롯 수정, 컨테이너 삭제.
  • 그리드에서 항목을 마우스 오른쪽 버튼으로 클릭합니다: 수량 편집, 항목 제거, 컨테이너 지우기.
  • 다음을 위해 빈 그리드 슬롯을 마우스 오른쪽 버튼으로 클릭합니다: 항목 추가, 컨테이너 지우기.
  • 다음에 대한 데이터가 있는 Pal 슬롯을 마우스 오른쪽 버튼으로 클릭하세요: Boss/Lucky/Awakened/DNA 토글, 잠금 레벨(4레벨), Max All Stats, Learn All Moves, Learned Skills, Bulk Sync Pal, 삭제.
  • 다음을 위해 빈 Pal 슬롯을 마우스 오른쪽 버튼으로 클릭하세요. 새 Pal 추가.

고급 사용자 팁

Base Pals의 Pal 슬롯 위로 마우스를 가져가면 정보 패널에서 해당 통계를 미리 볼 수 있습니다. 하단의 페이지 매김은 작업 중인 Pals의 현재 페이지를 보여줍니다. 탐색하려면 이전/다음 버튼이나 스크롤 휠을 사용하세요.
