Wiki 참조

문서 탭은 Palworld 게임 데이터를 앱 내에서 참조할 수 있는 가이드입니다. 도구를 떠나지 않고 팔, 아이템, 건축물, 스킬, 속성, 작업 적성, 기술을 탐색할 수 있습니다.

카테고리
  • 팔 — 완전한 팔 데이터베이스. 능력치, 속성 타입, 작업 적성, 파트너 스킬, 습득 스킬셋(레벨업 & 알 기술) 포함.
  • 아이템 — 모든 아이템의 카테고리, 희귀도, 가격, 무게, 최대 스택, 설명.
  • 건축물 — 구조물 데이터. 능력치, 등급, 재료, 거점 요구사항 포함.
  • 액티브 스킬 — 스킬 목록. 속성, 위력, 쿨타임, 타격 위력률, 최대 타격 수, 타격 간격을 게임 블루프린트에서 추출.
  • 패시브 스킬 — 패시브 스킬 데이터베이스. 등급, 설명, 효과 값.
  • 기술 — 기술 트리. 비용, 요구 레벨, 해제 아이템/건축물.
  • 속성 — 속성 참조. 색상 코드 및 관련 팔.
  • 작업 적성 — 작업 유형 참조 및 각 작업을 수행하는 팔.
조작 방법
  • 왼쪽 사이드바에서 카테고리를 선택하여 항목을 탐색합니다.
  • 목록의 항목을 클릭하면 오른쪽 패널에 세부 정보가 표시됩니다.
  • 검색창을 사용하여 이름 또는 내부 코드명으로 필터링합니다.

참고

모든 게임 데이터는 최신 익스포트에서 가져옵니다. 카테고리별로 필요 시에만 로드되어 빠른 시작이 가능합니다. 타격 데이터(WazaPowerRate, MaxHitNum, HitInterval)는 게임 블루프린트 파일에서 직접 추출됩니다.
