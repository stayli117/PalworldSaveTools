기능 개요

기본 마커(집 아이콘), 플레이어 마커(사람 아이콘) 및 제외 구역을 표시하는 대화형 세계 지도입니다. 마커를 클릭하여 기지/플레이어를 검사하고, 마우스 오른쪽 버튼을 클릭하여 관리 작업을 수행하고, 사이드바 트리를 사용하여 길드별로 탐색하세요.

기본 조작(왼쪽 클릭 및 버튼)

  • 마커를 클릭하여 선택하세요(글로우 효과로 강조 표시).
  • 마우스 휠을 스크롤하여 확대/축소합니다(단계당 1.15x, 범위 1.0x~30.0x).
  • 빈 공간을 클릭하고 드래그하여 이동하세요.
  • 툴바를 통해 지도 오버레이를 전환합니다: 기지, 플레이어, 반경 링 및 제외 구역 버튼.
  • 세계 지도와 트리 지도 보기 간에 전환합니다.
  • 검색창은 길드나 플레이어 이름으로 마커를 필터링합니다.
  • 왼쪽에 있는 베이스/플레이어 탭 버튼은 베이스 트리와 플레이어 트리 패널 사이를 전환합니다.
  • 정확한 좌표 정렬을 위해 보정 모드로 들어가려면 보정을 클릭하세요.

고급 단축키(더블 클릭 상호 작용)

  • 기본 또는 플레이어 마커를 두 번 클릭하면 지도 보기가 26.0x 확대/축소로 확대되고 해당 마커에 애니메이션이 적용되는 동시에 세부 정보로 정보 패널이 업데이트됩니다.
  • 구역 항목(지도에 그려진 제외 구역)을 더블클릭하면 이름 바꾸기 대화상자가 열립니다.
  • Rect가 선택된 영역 그리기 모드에서: 먼저 두 번 클릭하여 모서리 A를 설정하고, 두 번째 두 번 클릭하여 모서리 B를 설정하고 영역을 만듭니다.
  • 최소 3개의 점이 있는 다각형 모드: 두 번 클릭하면 다각형 영역이 닫히고 생성됩니다.

숨겨진 메뉴(마우스 오른쪽 버튼 클릭 컨텍스트)

  • 베이스 삭제, 베이스 내보내기, 반경 조정에 대한 베이스 마커를 마우스 오른쪽 버튼으로 클릭합니다.
  • 플레이어 삭제, 플레이어 이름 바꾸기, 보기 케이지 잠금 해제, 기술 잠금 해제에 대한 플레이어 마커를 마우스 오른쪽 버튼으로 클릭합니다.
  • 오른쪽 클릭 빈 지도 공간 for: 베이스 가져오기, 영역 그리기, 모든 영역 지우기, 영역 내보내기, 영역 가져오기.
  • 길드 수준 작업(길드 이름 바꾸기, 길드 삭제, 길드 기지 내보내기/가져오기, 컨테이너 ID 업데이트)을 포함한 추가 작업을 보려면 기본/플레이어 트리에서 항목을 마우스 오른쪽 버튼으로 클릭하세요.
  • 지도에서 zone을 마우스 오른쪽 버튼으로 클릭하여 구역 삭제(확인 포함), 구역 이름 바꾸기를 수행합니다.

고급 사용자 팁

영역 그리기 모드에서 Escape을 누르면 취소하고 즉시 종료됩니다. 알려진 위치를 마우스 왼쪽 버튼으로 클릭하고 실행 취소하려면 마우스 오른쪽 버튼을 클릭하여 보정 모드(보정 버튼)를 사용하여 지도를 게임 좌표와 정확하게 정렬하세요.
