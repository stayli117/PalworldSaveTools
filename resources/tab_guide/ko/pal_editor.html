기능 개요

모든 플레이어가 소유한 Pal을 위한 심층 편집 인터페이스. 통계, IV, 활성 스킬, 패시브 특성, 업무 적합성, 순위, 영혼 업그레이드, 외모 플래그(보스/럭키/각성/DNA) 등을 완전히 제어하여 파티(활성 분대) 및 Palbox(저장소)로 구성된 Pal을 찾아보세요. 오른쪽의 정보 패널에는 편집 가능한 모든 세부정보가 표시됩니다.

기본 조작(왼쪽 클릭 및 버튼)

  • 선수 선택...을 클릭하여 검색 가능한 팝업에서 선수를 선택합니다(필터링하려면 입력, 선택하려면 Enter).
  • 파티 또는 Palbox 슬롯을 클릭하여 편집할 Pal을 선택하세요. 정보 패널은 편집 가능한 모든 필드로 채워집니다.
  • 정보 패널 헤더에서 성별, 보스 알파, 행운/빛나는, 각성, 수입/DNA, 최대 모든 통계 등 토글 버튼을 클릭합니다.
  • 잠금 수준(0~3)을 순환하려면 즐겨찾는 별표를 클릭하세요.
  • 입력 대화 상자를 통해 인라인으로 편집하려면 이름(닉네임), 수준(1~80), 신뢰 순위(0~10) 등 레이블 필드를 클릭하세요.
  • IV 라벨(HP/공격/방어)을 클릭하여 0~100을 설정합니다.
  • 소울 라벨(HP/공격/방어/제작 속도)을 클릭하여 0~20을 설정합니다.
  • 활성 스킬 슬롯을 클릭하여 스킬 선택기를 엽니다.
  • 패시브 슬롯을 클릭하여 패시브 선택기를 엽니다.
  • 작업 적합성 값을 클릭하여 수준(0~10)을 설정합니다.
  • ← 이전 / 다음 → 버튼이나 그리드의 마우스 휠을 사용하여 Palbox 페이지 간을 이동합니다.

고급 단축키(더블 클릭 상호 작용)

  • Pal이 있는 Party 또는 Palbox 슬롯을 두 번 클릭하여 빠른 삭제(확인 없음 — 상황에 맞는 메뉴와 동일한 'delete_direct' 작업 실행).
  • 빈 Party 또는 Palbox 슬롯을 두 번 클릭하여 새 Pal 추가 생성 대화 상자를 엽니다.
  • 새 친구 추가 목록에서 친구를 더블클릭하면 즉시 생성됩니다.
  • 최대화를 전환하려면 FramelessDialog 제목 표시줄을 두 번 클릭하세요.

숨겨진 메뉴(마우스 오른쪽 버튼 클릭 컨텍스트)

  • Pal이 있는 파티 또는 Palbox 슬롯을 마우스 오른쪽 버튼으로 클릭하여 다음을 수행할 수 있습니다: Boss Alpha 토글, Lucky/Shiny 토글, Awakened 토글, Imported/DNA 토글, 잠금 레벨(하위 메뉴: 4레벨), Max All Stats, Learn All Moves, Learned Skills 대화 상자, Bulk Sync Pal, 삭제.
  • 다음을 위한 빈 슬롯을 마우스 오른쪽 버튼으로 클릭하세요. 새 Pal.
    • 추가
  • 동일한 확장된 컨텍스트 메뉴를 보려면 정보 패널에서 Pal 초상화를 마우스 오른쪽 버튼으로 클릭하세요.

고급 사용자 팁

키보드 단축키를 사용하면 편집 속도가 빨라집니다. Q = 이전 Palbox 페이지, E = 다음 Palbox 페이지, F = 초점을 맞추고 정보 패널로 스크롤, C = 스킬 보기를 전환합니다. 슬롯 위에 마우스를 올리면 슬롯을 선택하지 않고도 정보 패널에서 Pal을 미리 볼 수 있습니다. 대량 동기화(컨텍스트 메뉴에서)을 사용하여 한 친구의 통계/스킬/패시브를 한 번에 여러 다른 친구에게 복사할 수 있습니다.
