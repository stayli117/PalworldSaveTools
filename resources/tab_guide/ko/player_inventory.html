기능 개요

3개의 하위 탭이 있는 전체 플레이어 인벤토리 관리: 인벤토리(메인 가방의 모든 아이템/장비), 핵심 아이템(퀘스트 아이템, 조각상, 기술) 및 Stats(플레이어 속성: 레벨, HP, 체력, 공격, 방어, 작업 속도, 무게). 오른쪽의 장비 패널에는 무기, 액세서리, 음식, 갑옷, 방패, 글라이더 및 모듈 슬롯이 표시됩니다.

기본 조작(왼쪽 클릭 및 버튼)

  • 선수 선택...을 클릭하여 검색 가능한 팝업에서 선수를 선택하세요(필터링하려면 입력, 선택하려면 Enter).
  • 보기를 전환하려면 하위 탭을 클릭하세요.
  • 통계 하위 탭에서 각 통계 옆에 있는 +/− 버튼을 클릭하여 1씩 조정하거나 숫자 필드에 직접 값을 입력하고 Enter를 누르세요.
  • 레벨 +/− 버튼을 클릭하거나 레벨(1-80)을 입력하세요.
  • 인벤토리 탭에서 모든 지도 잠금 해제 + 빠른 이동 버튼을 클릭하면 모든 빠른 이동 지점이 표시됩니다.
  • 잠긴 장비 슬롯(무기/방어구/음식/액세서리)을 클릭하여 잠금 해제를 시도하세요. 툴팁에 어떤 아이템이 필요한지 알려줍니다.

고급 단축키(더블 클릭 상호 작용)

  • 인벤토리 또는 주요 항목 그리드의 항목을 두 번 클릭하여 확인 메시지 없이 직접 삭제합니다.
  • 빈 슬롯을 두 번 클릭하여 항목 추가 대화상자를 엽니다.
  • 항목이 있는 장비 슬롯을 두 번 클릭하여 해당 항목을 제거합니다. 빈 장비 슬롯을 두 번 클릭하여 해당 슬롯에 대한 항목 추가 선택기를 엽니다.

숨겨진 메뉴(마우스 오른쪽 버튼 클릭 컨텍스트)

  • 수량 편집, 삭제, 항목 추가를 위해 채워진 인벤토리 슬롯을 마우스 오른쪽 버튼으로 클릭합니다.
  • 빈 슬롯을 마우스 오른쪽 버튼으로 클릭하여 항목 추가, 슬롯 지우기(손상된 슬롯에 유용).
  • 수량 편집(음식 슬롯만), 슬롯 지우기 또는 항목 추가를 위해 장비 슬롯을 마우스 오른쪽 버튼으로 클릭합니다.
  • 항목 선택기 대화 상자에서는 검색 결과를 두 번 클릭하여 항목을 즉시 추가할 수도 있습니다.

고급 사용자 팁

항목 위에 마우스를 올리면 항목 이름, 수량, 내부 ID 및 전체 게임 설명이 포함된 도구 설명이 표시됩니다. 정렬 버튼은 항목을 카테고리별로 정리합니다. 주요 항목 탭에는 대량 작업을 위한 전용 "모든 조각상 추가" 및 "모든 주요 항목 추가" 버튼이 있습니다.
