기능 개요

저장 파일 관리를 위한 중앙 허브 — JSON과 SAV, GamePass와 Steam 형식 간 변환, 저장 간 문자 전송, 손상된 호스트 저장 수정, 플레이어 슬롯 삽입. 각 도구는 아이콘과 설명이 포함된 클릭 가능한 카드로 표시됩니다.

기본 조작(왼쪽 클릭 및 버튼)

  • 작업을 시작하려면 도구 카드를 클릭하세요(저장 변환, GamePass 변환→Steam, SteamID 변환, 맵 복원, 슬롯 인젝터, 저장 수정, 캐릭터 전송, 호스트 저장 수정).
  • 헤더의 Load Save 버튼을 이용하거나 메뉴 → 파일 → Load Save 경로를 이용하세요.
  • 제목 아래의 저장 경로 레이블은 파일 탐색기를 현재 저장 위치로 엽니다.
  • 상태 표시줄의 빠른 탐색 아이콘(플레이어, 길드, 기지, 친구)은 해당 탭으로 바로 이동합니다.

고급 단축키(더블 클릭 상호 작용)

  • 도구 카드에는 두 번 클릭 처리가 없습니다. 한 번의 클릭이 유일한 트리거입니다.
  • 변환 옵션 대화 상자(SAV‐JSON)에서는 목록 항목에 대한 더블 클릭이 지원되지 않습니다. 대신 옵션 버튼을 사용하세요.

숨겨진 메뉴(마우스 오른쪽 버튼 클릭 컨텍스트)

  • 도구 카드에는 오른쪽 클릭 동작이 없습니다. 모든 기능은 왼쪽 클릭만 가능합니다.
  • 그러나 메뉴 버튼(왼쪽 상단)에는 카드로 사용할 수 없는 많은 정리 기능이 포함되어 있습니다: 빈 길드 삭제, 비활성 기지/플레이어 삭제, 중복 플레이어 삭제, 불법 친구 수정, 유효하지 않은 아이템/구조물/친구 제거, 던전/석유 장비/침략자/보급품 재설정, 모든 구조물 수리, 개인 상자 잠금 해제 및 더.

고급 사용자 팁

.sav 파일을 창에 끌어서 놓으면 즉시 로드됩니다. 확인을 위해 녹색 놓기 영역 오버레이가 나타납니다. 변환 도구를 사용하기 전에 항상 백업하십시오. 메뉴 → 변경 사항 저장.

을 통해 정기적으로 변경 사항을 저장합니다.
