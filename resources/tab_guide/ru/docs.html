Справочник Wiki

Вкладка Документы предоставляет встроенный справочник по игровым данным Palworld. Просматривайте категории, чтобы изучать Палов, Предметы, Здания, Навыки, Элементы, Рабочие Навыки и Технологии, не покидая инструмент.

Категории Wiki
  • Палы — Полная база палов с характеристиками, типами элементов, рабочими навыками, навыками напарника и наборами изучаемых навыков (повышение уровня и яйца).
  • Предметы — Все предметы с категорией, редкостью, ценой, весом, размером стопки и описаниями.
  • Здания — Данные структур, включая характеристики, ранг, материалы и требования к базе.
  • Активные Навыки — Список навыков с элементом, силой, перезарядкой, коэффициентом силы удара, максимальным количеством ударов и интервалом, извлечёнными из игровых блюпринтов.
  • Пассивные Навыки — База пассивных навыков с рангом, описанием и значениями эффектов.
  • Технологии — Дерево технологий со стоимостью, требуемым уровнем и открываемыми предметами/зданиями.
  • Элементы — Справочник элементов с цветовыми кодами и связанными палами.
  • Рабочие Навыки — Справочник типов работ и какие палы выполняют каждый тип.
Навигация
  • Выберите категорию на левой боковой панели для просмотра записей.
  • Нажмите на запись в списке, чтобы увидеть её детали на правой панели.
  • Используйте строку поиска для фильтрации записей по имени или внутреннему коду.

Примечание

Все игровые данные получены из последних экспортов игры. Данные загружаются по требованию для каждой категории для быстрого запуска. Данные ударов (WazaPowerRate, MaxHitNum, HitInterval) извлекаются непосредственно из файлов блюпринтов игры.
