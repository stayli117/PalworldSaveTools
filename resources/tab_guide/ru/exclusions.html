Функциональный обзор

Три расположенные рядом панели поиска, отображающие исключенные UID игроков, идентификаторы гильдий и базовые идентификаторы. Перечисленные здесь элементы защищены от операций удаления и очистки.

Основные операции (щелкните левой кнопкой мыши и кнопки)

  • Нажмите любую запись об исключении, чтобы выбрать ее.
  • Введите строку поиска, чтобы фильтровать идентификаторы исключений в режиме реального времени.
  • Добавьте записи через контекстное меню, вызываемое правой кнопкой мыши на вкладках «Игроки», «Гильдии» или «Базы».
  • Сохранить и загрузить списки исключений через Меню → Исключения → Сохранить исключения и Загрузить исключения.

Расширенные ярлыки (взаимодействие с двойным щелчком мыши)

  • Двойной щелчок по строке исключения выдает сигнал item_double_clicked.
  • Обработчик по умолчанию не подключен — двойной щелчок в настоящее время не оказывает видимого влияния ни на одну из трех панелей исключений.

Скрытые меню (контекст, вызываемый правой кнопкой мыши)

Нажмите правой кнопкой мыши любую запись исключения для: Удалить исключение — удаляет выбранный идентификатор из списка защиты, делая его снова доступным для операций очистки.

Совет для опытных пользователей

Создайте список исключений перед выполнением любых функций массовой очистки. Исключения можно добавлять из контекстного меню на вкладках «Игроки», «Гильдии» и «Базы» — здесь не нужно вручную вводить идентификаторы. Чаще сохраняйте список исключений с помощью кнопки «Меню».
