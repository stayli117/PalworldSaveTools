Функциональный обзор

Интерактивная карта мира с указанием базовых маркеров (значок дома), маркеров игрока (значок человека) и запретных зон. Нажимайте маркеры, чтобы проверять базы/игроков, щелкните правой кнопкой мыши для действий по управлению и используйте дерево боковой панели для навигации по гильдии.

Основные операции (щелкните левой кнопкой мыши и кнопки)

  • Нажмите на маркер, чтобы выбрать его (выделится эффектом свечения).
  • Прокрутите колесо мыши для масштабирования (1,15x за шаг, диапазон 1,0x–30,0x).
  • Нажмите и перетащите пустое место для панорамирования.
  • Переключение наложений карты с помощью панели инструментов: кнопки «Базы», «Игроки», «Радиусные кольца» и «Зоны отчуждения».
  • Переключение между режимами просмотра карты мира и древовидной карты.
  • Панель поиска фильтрует маркеры по имени гильдии или игрока.
  • Кнопки вкладки «Базы/Игроки» слева переключают панели базового дерева и дерева игроков.
  • Нажмите Калибровать, чтобы войти в режим калибровки для точного выравнивания координат.

Расширенные ярлыки (взаимодействие с двойным щелчком мыши)

  • Дважды щелкните любой маркер базы или игрока, чтобы увеличить масштаб и анимировать вид карты для этого маркера с увеличением 26,0x, одновременно обновляя информационную панель с его подробностями.
  • Дважды щелкните элемент зоны (зона отчуждения, нарисованная на карте), чтобы открыть диалоговое окно переименования.
  • В режиме рисования зон с выбранным прямоугольником: первый двойной щелчок устанавливает угол A, второй двойной щелчок устанавливает угол B и создает зону.
  • В режиме «Многоугольник» с минимум 3 точками: двойной щелчок закроет и создаст зону многоугольника.

Скрытые меню (контекст, вызываемый правой кнопкой мыши)

  • Щелкните правой кнопкой мыши базовый маркер , чтобы: «Удалить базу», «Экспортировать базу», «Отрегулировать радиус».
  • Нажмите правой кнопкой мыши маркер игрока, чтобы: удалить игрока, переименовать игрока, разблокировать клетку просмотра, разблокировать технологии.
  • Нажмите правой кнопкой мыши пустое место на карте, чтобы: «Импорт базы», «Нарисовать зоны», «Очистить все зоны», «Экспорт зон», «Импорт зон».
  • Нажмите правой кнопкой мыши по элементам в дереве базы/игрока, чтобы выполнить дополнительные действия, включая операции на уровне гильдии (переименовать гильдию, удалить гильдию, экспортировать/импортировать базы для гильдии, обновить идентификаторы контейнеров).
  • Щелкните правой кнопкой мыши zone на карте, чтобы: Удалить зону (с подтверждением), Переименовать зону.

Совет для опытных пользователей

Нажмите Escape в режиме рисования зон, чтобы отменить операцию и немедленно выйти. Используйте режим калибровки (кнопка «Калибровать»), чтобы точно совместить карту с игровыми координатами, щелкнув левой кнопкой мыши известные местоположения и щелкнув правой кнопкой мыши, чтобы отменить действие.
