Функциональный обзор

Полное управление инвентарем игрока с тремя дополнительными вкладками: Инвентарь (все предметы/снаряжение в основной сумке), Ключевые предметы (квестовые предметы, изображения, технологии) и Статистика (атрибуты игрока: уровень, HP, выносливость, атака, защита, скорость работы, вес). На панели оборудования справа показаны слоты для оружия, аксессуаров, еды, брони, щита, планера и модулей.

Основные операции (щелкните левой кнопкой мыши и кнопки)

  • Нажмите Выбрать игрока..., чтобы выбрать игрока из всплывающего окна с возможностью поиска (введите текст для фильтрации, Enter для выбора).
  • Нажимайте вложенные вкладки, чтобы переключать представления.
  • На подвкладке «Статистика» нажмите кнопки +/- рядом с каждым показателем, чтобы изменить его на 1, или введите значение непосредственно в числовое поле и нажмите Enter.
  • Нажмите кнопки Уровень +/- или введите уровень (1–80).
  • Нажмите кнопку Разблокировать всю карту + быстрое путешествие на вкладке «Инвентарь», чтобы открыть все точки быстрого перемещения.
  • Нажмите заблокированные слоты для оборудования (оружие/броня/еда/аксессуары), чтобы попытаться разблокировать — всплывающая подсказка сообщит вам, какой предмет нужен.

Расширенные ярлыки (взаимодействие при двойном щелчке)

  • Дважды щелкните любой предмет в таблице «Инвентарь» или «Ключевые предметы», чтобы удалить его напрямую без запроса подтверждения.
  • Дважды щелкните пустой слот, чтобы открыть диалоговое окно добавления элемента.
  • Дважды щелкните слот оборудования, в котором есть предмет, чтобы удалить этот предмет; дважды щелкните пустой слот для оборудования, чтобы открыть средство выбора предметов для этого слота.

Скрытые меню (контекст, вызываемый правой кнопкой мыши)

  • Щелкните правой кнопкой мыши по заполненному слоту инвентаря, чтобы: «Редактировать количество», «Удалить», «Добавить элемент».
  • Щелкните правой кнопкой мыши по пустому слоту: «Добавить предмет», «Очистить слот» (полезно для поврежденных слотов).
  • Щелкните правой кнопкой мыши по слоту оборудования, чтобы: «Изменить количество» (только для слотов с едой), «Очистить слот» или «Добавить предмет».
  • Диалоговое окно выбора элемента также поддерживает двойной щелчок по результату поиска, чтобы немедленно добавить элемент.

Совет для опытных пользователей

Наведите курсор на любой предмет, чтобы увидеть всплывающую подсказку с названием предмета, количеством, внутренним идентификатором и полным описанием игры. Кнопка «Сортировать» упорядочивает элементы по категориям. На вкладке «Ключевые элементы» выделены кнопки «Добавить все изображения» и «Добавить все ключевые элементы» для массовых операций.
