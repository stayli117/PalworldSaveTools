Функциональный обзор

Таблица поиска всех игроков в сохранении. В столбцах показано имя игрока, последний раз его видели, уровень, количество друзей, UID, название гильдии, идентификатор гильдии и уровень гильдии. Используйте кнопки массовых действий внизу для управления предметами, приятелями и технологиями между несколькими игроками.

Основные операции (щелкните левой кнопкой мыши и кнопки)

  • Введите строку поиска, чтобы фильтровать игроков в режиме реального времени — будет отображено любое совпадение по столбцу.
  • Нажмите строку, чтобы выбрать ее.
  • Нажмите Массовое управление предметами, чтобы открыть диалоговое окно «Действие с предметом игрока» (добавление/удаление предметов у выбранных игроков).
  • Нажмите Массовое управление Pal для операций Pal.
  • Нажмите Массовое управление технологиями, чтобы разблокировать технологию.

Расширенные ярлыки (взаимодействие с двойным щелчком мыши)

  • Двойной щелчок по строке игрока выдает сигнал item_double_clicked.
  • Однако в текущем коде не подключен обработчик по умолчанию — двойной щелчок в настоящее время не имеет видимого эффекта.
  • (Предназначено для использования в будущем.)

Скрытые меню (контекст, вызываемый правой кнопкой мыши)

  • Нажмите правой кнопкой мыши строку игрока, чтобы: добавить/удалить исключение, удалить игрока, переименовать игрока, разблокировать клетку просмотра, сбросить временную метку, обновить идентификаторы контейнеров, разблокировать все технологии, изменить технические очки, изменить статистику игрока, установить уровень игрока.
  • Также включает действия на уровне гильдии для гильдии игрока: сделать лидером гильдии, удалить гильдию, переименовать гильдию, разблокировать все лабораторные исследования, установить уровень гильдии, импортировать базу в гильдию.

Совет для опытных пользователей

Сортируйте любой столбец, щелкнув его заголовок. Столбец UID (индекс столбца 4) используется в качестве ключа для операций исключения и удаления. Используйте Меню → Функции → Удалить неактивных игроков после проверки дат последнего посещения, чтобы массово удалить заброшенных игроков.
