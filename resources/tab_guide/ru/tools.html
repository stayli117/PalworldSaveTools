Функциональный обзор

Центральный центр управления файлами сохранений — конвертируйте форматы JSON и SAV, GamePass и Steam, переносите персонажей между сохранениями, исправляйте поврежденные сохранения хоста и добавляйте слоты игроков. Каждый инструмент представлен в виде кликабельной карточки со значком и описанием.

Основные операции (щелкните левой кнопкой мыши и кнопки)

  • Нажмите любую карточку инструмента, чтобы начать ее работу («Преобразовать сохранения», «Преобразовать GamePass→Steam», «Преобразовать SteamID», «Восстановить карту», «Инжектор слотов», «Изменить сохранение», «Перенос персонажей», «Исправить сохранение хоста»).
  • Используйте кнопку Загрузить сохранение в заголовке или Меню → Файл → Загрузить путь сохранения.
  • Метка пути сохранения под заголовком открывает проводник к текущему месту сохранения.
  • Значки быстрой навигации на панели статистики (Игроки, Гильдии, Базы, Друзья) переходят непосредственно на эти вкладки.

Расширенные ярлыки (взаимодействие с двойным щелчком мыши)

  • На карточках инструментов нет обработки двойного щелчка — единственным триггером является одинарный щелчок.
  • В диалоговом окне «Параметры преобразования» (SAV↔JSON) двойной щелчок по элементам списка не поддерживается — вместо этого используйте переключатели.

Скрытые меню (контекст, вызываемый правой кнопкой мыши)

  • На карточках инструментов нет действий, вызываемых щелчком правой кнопкой мыши — все функции выполняются только щелчком левой кнопкой мыши.
  • Однако кнопка Меню (вверху слева) содержит множество функций очистки, недоступных в виде карточек: удаление пустых гильдий, удаление неактивных баз/игроков, удаление дубликатов игроков, исправление нелегальных друзей, удаление недействительных предметов/структур/приятелей, сброс подземелий/нефтяной вышки/захватчика/припасов, ремонт всех построек, разблокировка привата. Сундуки и многое другое.

Совет для опытных пользователей

Перетащите файл .sav в окно, чтобы мгновенно загрузить его — для подтверждения появится зеленая зона перетаскивания. Всегда делайте резервную копию перед использованием инструментов преобразования. Регулярно сохраняйте изменения через Меню → Сохранить изменения.
