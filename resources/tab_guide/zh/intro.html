欢迎使用 Palworld 保存工具 V2

这是 Palworld Save Tools V2 — 一个精简版本，专注于您最需要的核心功能。一切都设计得快速、直观且易于导航。

浏览下面的部分或使用上面的目录直接跳到任何指南。

快速入门
  • 单击 菜单 → 加载保存 或将 .sav 文件拖放到窗口上即可开始。
  • 使用选项卡（地图、工具、玩家、公会、基地等）来探索和编辑您的保存数据。
  • 右键单击几乎所有内容以执行上下文操作 - 删除、编辑、导出等等。
  • 通过菜单→保存更改定期保存您的更改。自动创建备份.
有什么可用
  • Map Viewer — 带有基地和玩家标记、区域绘制和校准的交互式世界地图。
  • Tools — 保存转换、字符传输、主机修复、插槽注入器和清理功能。
  • 基地库存 — 浏览和管理所有基地的物品和工作伙伴。
  • 玩家库存 — 完整库存、关键物品、统计数据和装备管理.
  • Pal Editor — 深度好友编辑：统计数据、IV、技能、被动、工作适合性等。
  • 玩家 — 包含散装物品、好友和技术管理的玩家表。
  • 行会 — 具有完整上下文操作的两面板行会和成员管理。
  • Bases — 具有导出、导入、克隆和半径调整功能的大本营表。
  • 排除对象 — 保护名单，以保护玩家、公会和基地免遭清理。
  • Wiki — 应用内游戏数据参考：帕鲁、物品、技能、建筑、元素等。
