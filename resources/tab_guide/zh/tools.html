功能概述

保存文件管理的中央枢纽 — 在 JSON 和 SAV、GamePass 和 Steam 格式之间进行转换、在保存之间传输字符、修复损坏的主机保存以及注入玩家插槽。每个工具都以可点击的卡片形式呈现，带有图标和说明。

基本操作（左键单击和按钮）

  • 单击任意工具卡即可启动其操作（转换保存、转换GamePass→Steam、转换SteamID、恢复地图、插槽注入器、修改保存、角色传输、修复主机保存）.
  • 使用标题中的 Load Save 按钮或 Menu → File → Load Save 路径.
  • 标题下方的保存路径标签将文件资源管理器打开到当前保存位置。
  • 统计栏中的快速导航图标（玩家、公会、基地、伙伴）直接跳转到这些选项卡。

高级快捷键（双击交互）

  • 工具卡上没有双击处理——单击是唯一的触发器。
  • 在“转换选项”对话框 (SAV↔JSON) 中，不支持双击列表项 — 请改用选项按钮。

隐藏菜单（右键单击上下文）

  • 工具卡没有右键单击操作 - 所有功能都只能左键单击。
  • 然而，Menu按钮（左上角）包含许多作为卡牌不可用的清理功能：删除空公会、删除不活跃的基地/玩家、删除重复的玩家、修复非法好友、删除无效的物品/结构/好友、重置地下城/石油钻机/入侵者/供应、修复所有结构、解锁私人宝箱，以及更多.

高级用户提示

将 .sav 文件拖放到窗口上即可立即加载 - 会出现一个绿色的拖放区域覆盖层以进行确认。始终在转换工具之前进行备份。通过 Menu → Save Changes.
